# Кубик-прыгун — Игра-платформер

Курсовая работа по дисциплине **«Технологии программирования»**.  
Игра-платформер с бесконечным генератором платформ, системой рекордов и визуальными эффектами.

## Особенности

- Управление стрелками **← → ↑** или клавишами **A / D / ↑**
- Прыжок выполняется **только при нажатии ↑**
- Бесконечная генерация платформ с постепенным увеличением сложности
- След за кубиком с эффектом затухания
- Автоматическое сохранение рекорда в `high_score.txt`
- Анимированный звёздный фон
- Экран проигрыша с возможностью перезапуска (`R`) или выхода (`ESC`)

## Как запустить

1. Убедитесь, что установлен **Python 3.8+**
2. Установите зависимости:

python main.py
pip install pygame
